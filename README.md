# AgeGate Popup

## Install
Install with [npm](https://www.npmjs.com/):
or
Install with [yarn](https://yarnpkg.com/)

```sh
$ npm install --save @vankosoft/agegate-popup
```

```sh
$ yarn add @vankosoft/agegate-popup
```

## Usage
```javascript
new Agegate({
    cookie: 'age_gate',
    body: 'Are you 18 or older?',
    cancelUrl: 'https://www.disney.bg'
});
```
